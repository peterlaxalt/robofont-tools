# Robofont (and Drawbot) Tools

Various Robofont extensions, tools, and scripts I use for font development and design.

## Extensions
- Reference Character Window: Simple window to show your current character in another font for reference 
- Rotator: Rotate and repeat paths (useful for designing asterisks, etc.)
- Copy glyphs: Simple right click menu for copying unicode or glyphs from the grid view
- Monospace glyphs: Select glyphs and set a fixed width for them
- Glyph bulk spacing: Set left/right sidebearing values for multiple glyphs at a time

## Scripts
- Add glyphs from array: What it says, I use it occasionally when looking through unicode.

## Drawbot
- Character Set Proofer Lang Grid: What I use for proofing fonts. All config at the top of the script.
